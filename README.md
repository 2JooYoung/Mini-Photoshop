## Mini-Photoshop

Python, Tkinter, Pillow 를 사용해서 만든 간단한 이미지 편집 프로그램입니다.  
이미지를 열어서 확대/축소, 회전, 반전, 밝기/어둡기 조절, 블러, 엠보싱, 흑백 변환 등을 해볼 수 있습니다.

---

## About

Mini-Photoshop 은 로컬에 있는 이미지를 불러와서  
메뉴를 통해 여러 가지 필터를 적용해 보는 데스크톱 앱입니다.

구현된 기능:

- 이미지 열기 / 저장
- 확대 / 축소
- 상하 반전 / 좌우 반전
- 임의 각도 회전
- 밝기 조절 (밝게 / 어둡게)
- 블러(Blur) 효과
- 엠보싱(Emboss) 효과
- 흑백(Grayscale) 변환

---

## Tech Stack

- Python  
- Tkinter (GUI)  
- Pillow (PIL)
